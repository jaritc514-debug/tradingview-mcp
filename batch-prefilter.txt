STEP 1 — LOCATION:
- Read Leg H and Leg L directly from Trend Summary table

- Discount (0-33%) / Equilibrium (33-66%) / Premium (66-100%)
- If Equilibrium: mark as NO TRADE but continue to Step 2 — do NOT stop

STEP 2 — DIRECTION:
- Read Trend and Status directly from Trend Summary table
- Do not perform independent pivot analysis — trust v18 output entirely
- Always read and record Trend and Status regardless of location
- If Trend = Sideways: mark as NO TRADE

STEP 3 — ACTION MATRIX:
- Cross reference Location + Trend + Status:

  Confirmed Bullish + Discount → LONG VALID ✓
  Confirmed Bearish + Premium → SHORT VALID ✓
  Unconfirmed Bullish + Discount → LONG VALID ⚠ (lower probability)
  Unconfirmed Bearish + Premium → SHORT VALID ⚠ (lower probability)
  Any trend + Equilibrium → NO TRADE (EQ)
  Sideways (any location) → NO TRADE (Sideways)
  Everything else → NO ACTION
